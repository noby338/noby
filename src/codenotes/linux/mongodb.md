### 数据库结构
- 库
  - 集合
    - 文档
- 默认的库
  - admin
    - 用户和权限
  - local
    - 在集群模式下，该库的集合不会复制
  - config
    - 分片设置

- 默认端口号27017

## 命令
### 库
- show dbs
- use 数据库
  - 使用数据库，如果不存在立即创建并切换，存储机制分为内存和磁盘，使用use存放在内存中，还没有在磁盘中，当库中存在集合时，才会自动持久化到磁盘
- db
  - 显示当前使用的数据库
- db.dropDatabase()
  - 删除当前库

### 集合
- db.createClollection(name)
  - 显式创建集合（创建集合的时候不创建库）
- show collections
  - 显式集合
- db.集合.drop()
  - 删除集合
